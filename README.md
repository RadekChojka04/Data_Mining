# Data Mining

## Report_1

### Feature Discretization (Iris dataset):

Evaluated and compared unsupervised discretization methods (Equal Width, Equal Frequency, K-means, and Manual) to transform continuous biological measurements into categorical data. Performance was validated against original species labels to determine the most effective separation technique.

### Principal Component Analysis (City Quality of Life dataset):

Performed PCA on a multidimensional dataset of global city metrics. The analysis includes data standardization, variance evaluation and interpreting loadings to identify the most significant indicators of urban quality of life.

## Report_2

### Comparative Classification Analysis (Pima Indians Diabetes dataset)

Built and evaluated multiple baseline models, including:
* k-Nearest Neighbors (k-NN)
* Naive Bayes
* Decision Trees
* Bagging & Random Forest
* Support Vector Machines (SVM)

Model Evaluation:
Used rigorous statistical validation techniques, including 10-fold Cross-Validation, Bootstrap and the .632+ estimator to assess generalization capabilities and minimize classification errors.
